
📊 Customer Behavior Analysis
🔍 Overview

This project analyzes customer purchasing behavior to uncover trends, patterns, and business insights. It covers the complete data analytics workflow—from data preprocessing to SQL analysis and dashboard visualization.

📁 Dataset
Contains customer transaction data including:
Gender, Age Group
Product Category & Items
Purchase Amount
Review Ratings
Subscription & Discount details
Data was cleaned and processed before analysis.

🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & EDA
MySQL – Data querying and analysis
Power BI – Interactive dashboard creation
Gamma – Presentation (PPT)

⚙️ Steps Performed
Data Loading & Cleaning (Python)
Loaded dataset using Pandas
Handled missing values and duplicates
Standardized column formats
Exploratory Data Analysis (EDA)
Analyzed customer demographics and purchase trends
Identified key KPIs like average purchase and ratings
SQL Analysis (MySQL)
Performed business-driven queries, including:
Revenue by gender
Discount impact on purchases
Top-rated and most purchased products
Customer segmentation (New, Returning, Loyal)
Subscription vs non-subscription behavior
Dashboard Creation (Power BI)
Built an interactive dashboard to visualize insights
Reporting & Presentation
Created final report and presentation using Gamma

📊 Dashboard

Key Highlights:

Total Customers: 3.9K
Average Rating: 3.75
Average Purchase: $59.76
Revenue & Sales by Category
Customer segmentation by Age Group & Subscription
Discount usage analysis

📈 Results & Insights
Female and male customers show different revenue contributions
Discounts significantly influence purchasing behavior
Clothing category generates the highest revenue and sales
Young adults contribute the most to overall revenue
Subscribed customers tend to spend more than non-subscribers
Loyal customers form a valuable segment for business growth

🧠 Key SQL Insights
Identified top 5 products with highest ratings
Calculated discount usage percentage per product
Segmented customers into New, Returning, Loyal
Found top products per category using window functions
Analyzed repeat buyers vs subscription behavior
